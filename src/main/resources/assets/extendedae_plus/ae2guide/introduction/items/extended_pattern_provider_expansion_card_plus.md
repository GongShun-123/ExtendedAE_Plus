# Super Extended Pattern Provider Expansion Card

<ItemImage id="extendedae_plus:extended_pattern_provider_expansion_card_plus" scale="4" />

The **Super Extended Pattern Provider Expansion Card** is an upgrade card for ExtendedAE's extended pattern providers. Each installed card unlocks one additional pattern page, making it easier to keep large automation setups organized in a single provider.

## Function & Effects

- **Extra pattern page:** Each card unlocks **1 additional page** of pattern slots.
- **Slot capacity:** Each page contains **36 pattern slots**.
- **Installation limit:** Up to **3 cards** can be installed in one supported provider.
- **Maximum capacity:** With 3 cards installed, an extended pattern provider can use **4 pages / 144 pattern slots** in total.

## Supported Devices

- Extended Pattern Provider
- Extended Pattern Provider Part

## Removal Protection

If removing an expansion card would make the remaining unlocked pages unable to hold the current patterns, the card cannot be removed. Move or remove patterns from the higher pages first, then try removing the card again.

> **Note:** This card only expands the pattern pages of ExtendedAE's extended pattern providers. It does not increase the pattern capacity of the vanilla AE2 Pattern Provider.
